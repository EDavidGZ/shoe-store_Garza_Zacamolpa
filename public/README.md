# shoesStore
## Introduccion
Proyecto de **Desarrollo web** elaborado con **React js**.

___
### Librerias utilizadas
* boxicons
```
npm install boxicons --save
```
* firebase
```
npm install --save firebase
```

* react-icons
```
npm install react-icons --save
```
* react-router-dom
```
npm i react-router-dom
```
* sweetalert2
```
npm i sweetalert2
```
